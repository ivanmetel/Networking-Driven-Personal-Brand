# 0.1. Core

## Назначение раздела

Здесь документируются **фундаментальные основы** организации хранилища: модель семейств документов, принципы, терминология, классификация. Это "конституция" репозитория.

**Ключевой вопрос:** Как организованы знания? Что значат термины?

## Содержимое раздела

| Документ | Описание |
|----------|----------|
| [document-families.md](document-families.md) | **Модель семейств документов (F0-F9)** — главный документ |
| [principles.md](principles.md) | Принципы организации хранилища |
| [glossary.md](glossary.md) | Глоссарий терминов |
| [taxonomy.md](taxonomy.md) | Классификация и теги |

## Что здесь размещается

- **Модель семейств** — структура таблицы 3×3 и метасемейства F0
- **Принципы** — фундаментальные правила организации
- **Терминология** — определения ключевых понятий
- **Классификация** — системы тегов и категорий

## Связь с FPF

Раздел реализует паттерны:
- **A.1.1 (Bounded Context)** — определяет контекст и терминологию
- **A.1 (Holonic Foundation)** — описывает иерархию знаний
