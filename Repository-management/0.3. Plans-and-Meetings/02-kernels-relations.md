## Типы связей

| Тип | Семантика | Пример |
|-----|-----------|--------|
| **создаёт** | X производит Y | Команда → Продукт |
| **поддерживает** | X обеспечивает работу Y | DevOps → Сервис |
| **использует** | X потребляет Y | App → API |
| **часть** | X является компонентом Y | Модуль → Система |
