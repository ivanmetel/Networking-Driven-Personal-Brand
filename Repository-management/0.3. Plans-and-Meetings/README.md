# 0.3. Plans-and-Meetings

## Назначение раздела

Здесь размещаются **планы работы** над хранилищем и **протоколы совещаний** по его развитию.

**Ключевой вопрос:** Что запланировано? Что обсудили?

## Что здесь размещается

- **Планы** — roadmap развития хранилища
- **Протоколы** — записи совещаний и решений
- **Ретроспективы** — анализ прошедших периодов
- **OKR/KPI** — цели и метрики хранилища

## Типичные документы

```
plans/
├── 2026-Q1-plan.md          # Квартальный план
├── 2026-01-08-meeting.md    # Протокол совещания
└── roadmap.md               # Общий roadmap
```

## Шаблон протокола совещания

```markdown
---
type: meeting
date: YYYY-MM-DD
participants: [participant1, participant2]
status: completed
---

# Совещание DD.MM.YYYY

## Повестка
1. [Тема 1]
2. [Тема 2]

## Обсуждение
### Тема 1
[Краткое содержание]

## Решения
- [ ] [Действие 1] — @ответственный
- [ ] [Действие 2] — @ответственный

## Следующее совещание
[Дата и время]
```
