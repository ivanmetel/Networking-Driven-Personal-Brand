# B2. System-of-Interest — Приложение

## Назначение

Документация о **Приложении** — нашей целевой системе.

## Определение

**Приложение** — система, поддерживающая создание и проведение ORA:
- Регистрация
- Билеты/платежи
- Коммуникации
- Данные и пост-ивент контур
- Интеграции

## Функциональные блоки

1. **Регистрация** — создание событий, приглашение участников
2. **Билеты/платежи** — продажа, возвраты, интеграции
3. **Коммуникации** — уведомления, напоминания, рассылки
4. **Данные** — аналитика, отчёты, экспорт
5. **Пост-ивент** — обратная связь, follow-up, сообщество
