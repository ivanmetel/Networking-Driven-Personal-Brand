# B3. Constructor — Planora

## Назначение

Документация о **Planora** — системе создания Приложения и его внедрения.

## Определение

**Planora** — система создания Приложения и организации его внедрения в практику организаторов ORA.

## Подразделы

| Код | Роль | Вопрос | Содержание |
|-----|------|--------|------------|
| B3.1 | Предприниматель | Зачем? | Миссия, бизнес-модель |
| B3.2 | Инженер | Как устроена? | ИТ-служба + ORA-студия |
| B3.3 | Менеджер | Как работает? | Процессы, команда |

## Две части Planora

### ИТ-служба

**Отвечает за Приложение:**
- Создание и развитие
- Эксплуатация
- Функциональность и надёжность
- Интеграции
- Данные и безопасность

### ORA-студия

**Отвечает за надсистему Приложения (ORA Organization):**
- Внедрение Приложения в практику
- Обучение организаторов
- Методики и шаблоны описания ORA
- Стандарты качества
- Сопровождение

## Связь частей

```
ИТ-служба создаёт Приложение (B2)
         ↓
ORA-студия внедряет в ORA Organization (B1)
         ↓
ORA Organization использует для создания ORA (A3)
         ↓
ORA воздействует на Впечатлённого человека (A2)
```
