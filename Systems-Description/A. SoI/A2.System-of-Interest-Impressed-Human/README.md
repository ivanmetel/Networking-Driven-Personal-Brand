# A2. System-of-Interest — Впечатлённый человек

## Назначение

Документация о **целевом состоянии человека**, которого мы хотим достичь через ORA.

## Определение

**Впечатлённый человек** — человек как система, переведённый в целевое состояние "впечатлённости" в контексте X и окне времени W.

**Целевая характеристика:** "впечатлённость" — это характеристика человека, не отдельная система.

## Подразделы

| Код | Роль | Вопрос | Содержание |
|-----|------|--------|------------|
| A2.1 | Предприниматель | Что это? | Определение впечатлённости, критерии успеха |
| A2.2 | Инженер | Из чего состоит? | Компоненты: эмоции, связи, знания |
| A2.3 | Менеджер | Как достигается? | Путь участника: до → во время → после |

## Измерение впечатлённости

- **Окно W** — состояние сразу после ORA
- **Окно W+Δ** — закрепление эффекта через время
- **Контекст X** — в какой области повышена впечатлённость
