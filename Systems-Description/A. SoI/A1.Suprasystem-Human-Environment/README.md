# A1. Suprasystem — Среда человека

## Назначение

Документация о **среде**, в которой живёт человек и которая влияет на его участие в ORA.

## Определение

**Среда человека** — устойчивые системы окружения, влияющие на:
- Доступ к ORA (время, деньги, логистика)
- Ожидания до ORA (что слышал, от кого)
- Закрепление впечатлённости после ORA (с кем обсудит, как применит)

## Компоненты среды

1. **Семья/домохозяйство** — время, бюджет, нормы
2. **Круг общения/сообщество** — смыслы, "зачем идти"
3. **Городская инфраструктура** — доступность, безопасность, транспорт
4. **Информационная среда** — ожидания, конкуренция за внимание
