# **Networking-Driven Personal Brand: [добавить слоган]**

Это хранилище знаний компании CooperActive Systems, организованное по SRD-методу с двухъядерной структурой.

## Цепочка создания ценности

```
CooperActive Systems → Приложение → Организатор ORA → ORA → Впечатлённый человек → Среда, в которой живёт человек (edit)
```

## Что такое Personal Brand

[**Personal Brand** — это системное управление впечатлением, которое вы оставляете в сознании других людей. Это то, что говорят о вас, когда вы выходите из комнаты (или после завершения ORA).

## Что такое Networking

**Networking** — это искусство создания и поддержания сети доверительных отношений. В рамках нашей системы нетворкинг является топливом для личного бренда, превращая случайные встречи в стратегические активы.] edit


## Два ядра хранилища

Хранилище организовано вокруг **двух системных ядер**:

| Ядро | Название | Целевая система | Надсистема | Система создания |
|------|----------|-----------------|------------|------------------|
| **A. SoI** | Целевая система | Впечатлённый человек | Среда, в которой живёт человек | ORA (ивент-продукт) |
| **B. Product** | Наша система | Приложение | Организатор ORA | Planora |

## Определения систем

### Ядро A: Целевая система

1. **Целевая система (A2)** — **Впечатлённый человек (Impressed Human).**
   Человек как система, которого ORA переводит в целевое состояние "впечатлённости" в контексте X и окне времени W.

2. **Надсистема (A1)** — **Среда, в которой живёт человек (Human Environment).**
   Устойчивые системы окружения: семья, круг общения, город, инфосреда.

3. **Система создания (A3)** — **ORA (Event-as-Product).**
   Мероприятие-как-продукт, воздействующее на человека и повышающее его впечатлённость.

### Ядро B: Наша система

4. **Наша система (B2)** — **Приложение (App).**
   Система, поддерживающая создание и проведение ORA: регистрация, билеты, коммуникации, данные.

5. **Надсистема для нашей системы (B1)** — **Организатор ORA (ORA Organization).**
   Система, которая проектирует и проводит ORA, используя Приложение как подсистему.

6. **Система создания нашей системы (B3)** — **Компания Planora.**
   Компания включает ИТ-службу (создаёт Приложение) и ORA-студию (внедряет в практику).
